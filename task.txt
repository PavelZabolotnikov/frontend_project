Адреса эндпоинтов для сущности "Пользователи":
Получить всех пользователей: GET /users
Получить пользователя по id: GET /users/{id}
Создать пользователя: POST /users
Изменить пользователя: PUT /users/{id}
Удалить пользователя: DELETE /users/{id}

Статус коды ответов:
Успешный код: 200 OK
Запрос выполнен успешно, и в результате был создан новый ресурс: 201 Created
Нет содержимого для отправки: 204 No Content
Код ошибки запроса: 400 Bad Request
Клиент не имеет прав доступа к контенту, то есть он неавторизован: 403 Forbidden
Сервер не может найти запрошенный ресурс: 404 Not Found
Код ошибки сервера: 500 Internal Server Error
